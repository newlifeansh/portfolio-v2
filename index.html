2025.09 — 현재
AI / LLM 게이미피케이션 리텐션 체류시간
P
문제 정의
카카오뱅크 투자 서비스의 낮은 일 방문율과 단순 정보 제공 방식의 한계. 고객이 서비스에 머무를 이유가 부족해 체류시간과 재방문율이 낮은 상태였다.
A
분석 및 가설
금융 정보 학습 니즈와 게이미피케이션의 결합 가설 수립. 퀴즈 형식이 반복 참여를 유도하고 체류시간과 리텐션을 동시에 끌어올릴 수 있는 구조임을 검증.
S
해결 방식
  • GPT 벡터 유사도 모델을 활용해 금융 키워드 퀴즈를 자동 생성하는 AI 파이프라인 설계
  • Claude Code 기반 퀴즈 생성기 구축으로 매일 신규 퀴즈 무인 운영 체계 구현
  • AdFit 광고 연동으로 퀴즈 참여 → 광고 노출 → 수익화 흐름 설계
  • 스트릭·리워드 구조로 재방문 동기 설계
R
성과
